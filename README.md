# 📊 Grade Manager (Java Console Application)

A simple console-based application developed in Java to manage student grades.  
This project was created as a learning exercise to practice object-oriented programming, collections, and user input
handling.

---

## 🚀 Features

- Add new grades (subject + score)
- View all grades
- Calculate average grade
- Display passed subjects
- Display failed subjects
- Delete grades
- Input validation for user entries

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes and objects
- Encapsulation (getters)
- ArrayList (Java Collections)
- Control flow (loops, conditionals, switch)
- User input handling with Scanner
- Basic error handling and validation

---

## 📂 Project Structure

- `Main.java` → Application entry point and menu logic
- `GestorNotas.java` → Core logic for managing grades
- `Nota.java` → Data model representing a grade

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/grade-manager-java.git

2. Navigate to the project directory:
   cd grade-manager-java

3. Compile the project:
   javac *.java

4. Run the application:
   java Main
